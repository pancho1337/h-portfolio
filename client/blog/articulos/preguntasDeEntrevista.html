Preguntas para entrevistas full-stack

La mayoría de las preguntas que te harán serán técnicas, en estos problemas estan analizando las estrategias que tu estas applicando para resolver tales problema.

HTML

¿Cuál es la importancia de Doctype en HTML?

Doctype le dice al navegador qué versión del estándar HTML / XHTML está usando  y cómo representar la página.


¿Cuál es la diferencia entre display: none y visibility: hidden?

visibility: hidden oculta el elemento, pero sigue conservando el espacio que ocupaba, los elementos adyacentes conservan su ubicación.

display:none quita el elemento del DOM. El espacio que ocupaba; se libera, y otros elementos adyacentes pueden ocupar su lugar. 

 

¿Cuál es la diferencia entre sessionStorage y localStorage?

sessionStorage solo está disponible cuando se abre la pestaña de un navegador.

localStorage sobrevive al cerrar y volver a abrir un navegador.


¿Qué todos los nuevos elementos de formulario se introdujeron en HTML5?

<datalist>: especifica una lista de opciones para los controles de entrada.

<keygen>: esta etiqueta genera una clave de cifrado.

<output>: define el resultado de una expresión.

¿Qué es microdata?

Microdata es una especificación HTML WHATWG utilizada para anidar metadatos dentro del contenido existente en páginas web. Los motores de búsqueda, los rastreadores web y los navegadores pueden extraer y procesar microdatos de una página web y usarlos para proporcionar una experiencia de navegación más rica para los usuarios.

CSS

¿Que es la diferencia entre em, px, pt y %?

CSS tiene cuatro tamaños de unidades diferentes que se puede usar para sus márgenes, bordes, fuentes, etc. Estos son píxeles (px), puntos (pt), ems (em) y porcentajes (%).Podemos separar estas unidades en dos grupos diferentes, fijos y relativos. Los píxeles (px) y los puntos (pt) son fijos, mientras que em y porcentajes (%) son tamaños de unidad relativos. Los tamaños de unidad relativos son realmente poderosos al crear diseños escalables. Las unidades de tamaño fijo son buenas para usar cuando siempre desea el mismo tamaño de elemento sin importar dónde lo coloque.


¿Qué es Specificity en CSS?

La especificidad es el medio por el cual los navegadores deciden qué valores de propiedad CSS son los más relevantes para un elemento y, por lo tanto, se aplicarán. La especificidad se basa en las reglas de coincidencia que se componen de diferentes tipos de selectores CSS.


¿Qué es un Pre-Processors en CSS?

El pre-processor en CSS es una herramienta que le permite crear código CSS mucho más rápido, de manera más estructurada. Los preprocesadores extienden el CSS funcional al agregar variables, mixins, parciales, también permiten usar operadores dentro del código. Algunos ejemplos son SAS, LESS y Stylus.


 ¿Qué es el box model?

Todos los elementos HTML se pueden considerar como cuadros. En CSS, el término "modelo de caja"(box model) se usa cuando se habla de diseño y layout.

El modelo de cuadro CSS es esencialmente un cuadro que envuelve todos los elementos HTML. Consiste en: márgenes, bordes, relleno y el contenido real.
